# SQL Exploratory Data Analysis Project

## Project Overview
This project performs Exploratory Data Analysis using SQL to understand business data and generate insights.

## Objectives
- Explore database structure
- Identify dimensions and measures
- Analyze key metrics
- Perform ranking analysis

## Tools
- SQL Server
- GitHub
- Excel

## Analysis Steps
1. Database Exploration
2. Dimension Exploration
3. Date Analysis
4. Measure Analysis
5. Magnitude Analysis
6. Ranking Analysis
